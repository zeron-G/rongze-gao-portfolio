# Rongze Gao Portfolio

A cinematic personal website for Rongze Gao, built with React, Vite, and Framer Motion.

## Highlights

- custom signal-field hero animation inspired by cockpit instrumentation
- scroll-driven storytelling for core projects and career trajectory
- live GitHub data pull with a researched fallback snapshot
- GitHub Pages deployment workflow included

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production build is configured for a GitHub Pages repository named `rongze-gao-portfolio`.
